HTML tags

<table>
Table tag is used to create simple tables.

Here are some elements used for table structure:

<table>
the table container
<caption>
the label/description of the table
<thead>
the header area of the table
<tbody>
the body of the table
<tr>
table row
<th>
table heading cell
<td>
table data cell
<form>
The <form> tag is used to create a form for any desired input.
Attribues used with <form> include but are not limited to:
<button>, <input>, <type>, <for>, <value>, <name>, <action> etc.
<div>
This tag is a block level tag by default. It can be used as a container. Div containers can be used for many things.
For example, we can create a Drop Down Menu.
Lists
We have 2 types of lists: ordered and unordered.
Unordered lists <ul>
Unordered lists can be used when the order of the items don't matter. Such as an ingredient list for a cake recipe.
Ordered lists<ol>
Ordered lists can be used when the order of the items matter. Such as the instructions of the very same recipe.
Links: anchor tag<a>
Anchor tags are used to add hyperlink to connect one page to another.
<href>
The attribute <href> can be used to assign an URL go a destinated page or even navigate inside one page. Without the <href> tag an anchor tag <a> is not an hyperlink.
