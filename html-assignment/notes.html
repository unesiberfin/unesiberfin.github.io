HTML tags

<table>
Table tag is used to create simple tables. Here are some elements used for table structure:
<div>
This tag is a block level tag by default. It can be used as a container. Div containers can be used for many things.
For example, we can create a Drop Down Menu.
<ul>
Unordered lists can be used when the order of the items don't matter. Such as an ingredient list for a cake recipe.
<ol>
Ordered lists can be used when the order of the items matter. Such as the instructions of the very same recipe.
<a>
Anchor tags are used to add hyperlink to connect one page to another.
